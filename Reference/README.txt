📁 Reference Folder – Setup Required
This folder is currently empty and must be initialized before running the pipeline.

📌 Setup Instructions
Please follow the guide for full details, or run the appropriate command below:

✅ Option 1 – From the project root:


bash modules/utils/setup_reference.sh -g hg38

✅ Option 2 – From inside this folder:

bash ../modules/utils/setup_reference.sh -g hg38



Replace hg38 with your desired genome (e.g., mm10, hg19, etc.).

> Run this only once per machine or environment.

# 📁 Spike-in Reference Folder – Setup Required

This folder is currently empty and **must be set up before running any spike-in–based normalization**.

## 📌 Setup Instructions

Please refer to the user guide for detailed information, or use one of the options below:

### ✅ Option 1 – From the **project root**:

bash modules/utils/setup_spikein_refs.sh


### ✅ Option 2 – From **inside this folder**:


bash ../modules/utils/setup_spikein_refs.sh




> ⚠️ This step downloads and indexes all required spike-in genomes (e.g., *dm6*, *sacCer3*, *ERCC*, etc.).
> Run this only once per machine or environment.
